# Academic Performance & Infrastructure Analytics

An interactive Power BI dashboard analyzing school performance, infrastructure, attendance, and teacher data using integrated datasets.

## 🚀 Live Dashboard
👉 https://app.powerbi.com/view?r=eyJrIjoiY2E2Y2Y3YTctZmRhNy00YmFlLWE1NDAtZTkzY2Q1MmFkODJhIiwidCI6ImE2Nzc2NDI4LWZiNmItNGJkNC1hOWM1LWVjY2U2OTIxYjA4MyJ9

## 📂 Data Sources
- Student Registry  
- School Registry  
- Teacher Registry  
- Assessment Dataset  
- Attendance Registry  

## 📈 Features
- KPI metrics (students, teachers, schools, PTR)  
- Performance analysis (district/block/school level)  
- Attendance tracking  
- Infrastructure insights  
- Correlation between attendance & performance  

## 🛠️ Tools Used
- Power BI  
- Excel  
- DAX  

## 🎯 Objective
To provide data-driven insights for improving school performance and decision-making.
