# 📚 Academic Performance & Infrastructure Analytics (Power BI)

## 🔍 Project Overview
An interactive Power BI dashboard analyzing school performance, infrastructure, attendance, and teacher data using integrated datasets. This dashboard helps stakeholders gain actionable insights for improving school operations and student outcomes.

## 🚀 Live Dashboard
[View Dashboard](https://app.powerbi.com/view?r=eyJrIjoiY2E2Y2Y3YTctZmRhNy00YmFlLWE1NDAtZTkzY2Q1MmFkODJhIiwidCI6ImE2Nzc2NDI4LWZiNmItNGJkNC1hOWM1LWVjY2U2OTIxYjA4MyJ9)

## 📂 Data Sources
- **Student Registry**  
- **School Registry**  
- **Teacher Registry**  
- **Assessment Dataset**  
- **Attendance Registry**  

## 🛠️ Tools Used
- Power BI  
- Microsoft Excel  
- DAX (Data Analysis Expressions)  

## 📈 Key Features
- KPI metrics: total students, teachers, schools, PTR (pupil-teacher ratio)  
- Academic performance analysis (district, block, school level)  
- Attendance tracking  
- Infrastructure insights  
- Correlation between attendance & performance  

## 🎯 Objective
To provide data-driven insights for improving school performance and supporting effective decision-making.

## 💡 Insights & Recommendations
- Identify high-performing and low-performing schools  
- Highlight infrastructure gaps affecting learning outcomes  
- Monitor attendance patterns and their impact on performance  
- Support targeted interventions for teachers and students  

## 🎯 Learning Outcomes
- Data integration from multiple school datasets  
- KPI and trend analysis in Power BI  
- Visual correlation of performance and attendance  
- Real-world educational analytics problem-solving
